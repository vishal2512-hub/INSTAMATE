# INSTAMATE

**A sleek Instagram media downloader built with React and Vite**

---

## 🔍 Description

INSTAMATE is a lightweight, client-side web app that enables users to download Instagram photos, videos, and reels through URL input. Developed with React and Vite, it offers a clean, user-friendly interface and quick performance.

---

## 🎯 Features

- Download Instagram content (images, videos, reels) by URL  
- Preview media before downloading  
- Clean, responsive UI built in React  
- Fast build with Vite + HMR support  
- Easily extendable and maintainable architecture

---

## 🧱 Tech Stack

- **Framework:** React (via Vite template)  
- **Bundler:** Vite  
- **Styling:** CSS / optional SCSS  
- **HTTP Requests:** `fetch` API
